## 🚀 Overview

A modern **MERN Stack Product Manager App** where users can:

✨ Add products
✏️ Edit products
🗑️ Delete products
🌙 Toggle Dark/Light Mode

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/Abhinav35807/mern-final-.git
cd mern-final-
```

---

### 2️⃣ Backend Setup

```
cd backend
npm install
npm start
```

---

### 3️⃣ Frontend Setup

```
cd frontend
npm install
npm start
```
